# Portfolio Victoria Arenque

Portfolio profissional de Victoria Arenque, Senior Product Designer.

## 🚀 Estrutura do Projeto

```
/
├── index.html      # Estrutura HTML principal
├── styles.css      # Estilos globais e componentes
├── script.js       # Funcionalidades JavaScript
└── README.md       # Documentação
```

## 🎨 Design System

### Cores
- **Background**: `#000000` (Preto)
- **Text Primary**: `#FFFFFF` (Branco)
- **Text Secondary**: `#999999` (Cinza)
- **Borders**: `#2A2A2A` (Cinza escuro)

### Tipografia
- **Fonte**: Inter (Google Fonts)
- **Pesos**: 400 (Regular), 500 (Medium), 600 (Semibold), 700 (Bold)

### Componentes Implementados

#### ✅ Header
- Logo/Nome à esquerda
- Botão CTA "Resume" à direita
- Fixed position no topo
- Totalmente responsivo

#### ✅ Hero Section
- Label "GET STARTED"
- Título "Selected Projects"
- Descrição personalizada
- Toggle Switch (Personal/Business)
- Centralizado e espaçado

#### ✅ Toggle Switch
- Opções: Personal | Business
- Estado ativo visual
- Funcionalidade JavaScript
- Transições suaves

#### ✅ Projects Grid
- Grid de 4 colunas: ID | Título | Brand | Habilidades
- 3 projetos de exemplo
- Hover effects sutis
- Bordas inferiores como separadores
- Clicável com interação JavaScript
- Totalmente responsivo (1 coluna no mobile)

## 📱 Responsividade

- **Desktop**: Layout completo com padding lateral de 48px
- **Tablet** (≤768px): Padding lateral de 32px e fontes ajustadas
- **Mobile** (≤480px): Padding lateral de 24px e layout otimizado para telas pequenas

## 🔧 Como Usar

1. Abra o arquivo `index.html` em qualquer navegador moderno
2. Ou use um servidor local como Live Server (VS Code)
