# Instruções para Adicionar a Foto

Para que o avatar funcione corretamente, você precisa adicionar a imagem da Victoria ao projeto.

## Passos:

1. Renomeie o arquivo de imagem para: `victoria-photo.jpg`
2. Coloque o arquivo na pasta: `/Users/varenque/Desktop/Experimentos/`
3. A imagem deve estar no mesmo diretório que os arquivos `index.html`, `styles.css`, etc.

## Estrutura esperada:

```
/Users/varenque/Desktop/Experimentos/
  - index.html
  - styles.css
  - script.js
  - victoria-photo.jpg  ← Adicione este arquivo
  - README.md
```

## Formatos suportados:
- `.jpg` ou `.jpeg`
- `.png`

Se preferir usar outro nome ou formato, atualize as referências no `index.html`:
- Linha 17: `<img src="victoria-photo.jpg" ...>`
- Linha 32: `<img src="victoria-photo.jpg" ...>`
